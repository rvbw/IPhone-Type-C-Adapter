# iPhone Type-C Adapter

This project is a 3D CAD design of an iPhone Type-C adapter created using Onshape.
It was developed as my first project during my training period.

## Purpose
- Practice CAD design fundamentals
- Apply accurate dimensions and tolerances
- Build a simple, functional adapter design

## Tools Used
- Onshape
- Export format: Parasolid (.x_t)
